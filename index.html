Quem Sou

Adoro codificar e proporcionar uma ótima experiência para usuários :D

Venho criando interfaces desde 2011. Comecei a programar no incio do conceito de tableless.

Comecei trabalhando somente com a parte gráfica dos projetos web. Criando a interface de sites e sistemas somente com Photoshop. Assim que me formei em design gráfico, decidi me aperfeiçoar na área de front-end, do qual me apaixonei por HTML, CSS e JS. Hoje, também, proucuro estudar e desenvolver para platafarmas mobile, trabalhando com sistemas híbridos, como Ionic.

Executar um projeto do início ao fim é um prazer, buscando entregar ao usuário um produto que o ajude a ter soluções eficientes, mantendo a interface para o usuário tão intuitiva quanto possível (UX / UI).
